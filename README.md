## Features

✨ **Beautiful Design**
- Animated gradient background
- Smooth scrolling navigation
- Responsive design (works on all devices)
- Floating hearts animation

📝 **Sections**
- **Home**: Hero section with call-to-action

🎨 **Design Elements**
- Gradient backgrounds
- Smooth animations and transitions

- Interactive cards
- Timeline visualization

## How to Use

1. Open `index.html` in your browser
2. Customize the messages in the HTML file with your own
3. Update the Gallery and Memories sections with your personal touches
4. Deploy to GitHub Pages or any hosting service

## Customization

### Change Messages
Edit the text in `index.html` in the Messages section to personalize it.

### Change Colors
Modify the color values in `style.css` to match your preference.

### Add Images
Replace the gradient backgrounds in Gallery items with actual images.

## Deploy to GitHub Pages

1. Push this code to your GitHub repository
2. Go to Settings → Pages
3. Select 'main' branch as source
4. Your site will be live at: `https://yourusername.github.io/Sakhsu`
